Documentation

The web application offers users comprehensive access to information about a wide range of movies, directors, and genres. Users can register for an account, update their personal information, and curate a personalized list of their favorite movies.

Endpoints

GET /movies Returns a list of all movies List of movies with title, description, genre, director, image URL, and whether it’s featured None None JSON
GET /movies/:title Returns data about a single movie by title Data about a specific movie with title, description, genre, director, image URL, and whether it’s featured title (string) None JSON
GET /genres/:name Returns data about a genre by name Data about a specific genre including its description name (string) None JSON
GET /directors/:name Returns data about a director by name Data about a specific director including bio, birth year, and death year name (string) None JSON
